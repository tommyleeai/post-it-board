## [1.3.1] - 2026-04-13

### ✨ 新增 (Added)
*   **系統更新日誌功能 (Changelog System)**：每次網站進行改版後，首次登入將自動彈出版本更新資訊，確保您能第一時間掌握最新的功能與調整。並在左上角標題旁顯示當前版本號，點擊可隨時查看歷史更新。
*   **多層次懸停選單 (Multi-level FAB Menu)**：優化了右下角的懸停發布群組。滑鼠移到主按鈕 `+` 時將觸發第一段「向上展開」；繼續將滑鼠移至印章圖示上，將觸發第二段「向左展開」，有效節省畫面干擾。
*   **交錯流體飛出動畫 (Staggered Silky Animation)**：印章選項套用特製的蘋果級快出慢停曲線 `cubic-bezier(0.2, 0.9, 0.2, 1)`，並配合 `0.05s` 的時間差依序交錯飛出，總耗時極限壓在 0.45 秒以內，帶來不拖泥帶水、極致高級的流體 UI 反饋手感。

### 🐛 修正與優化 (Fixed & Changed)
*   **印章預覽白底修復**：針對去背景的印章圖案（如「完成」、「准奏」），修復了它們在彈出選單預覽與游標跟隨階段，因為透明背景而導致圖示與軟木塞底圖重疊無法辨識的 Bug。現已皆附上不超出邊框範圍的實心純白底色。
*   **真實印泥覆蓋效果調整**：大幅優化了將印章蓋在便利貼時的物理視覺感，將印章尺寸暴力放大一倍，並讓按壓動畫最終落於 `0.45` 的半透明度，完美重現真實墨水印泥微透紙背，卻又霸氣十足的質感。

## [1.3.0] - 2026-04-12
> 註：詳細架構設計可參見 `progress_v1.3.md`

### ✨ 新增 (Added)
*   **歷史歸檔區 (Archive Manager)**：新增獨立的 Firestore 資料集合。蓋下「已完成」印章的便利貼不再被直接刪除，而是歸檔封存，並可隨時在歷史紀錄面板中檢視或重新復原。
*   **層疊樣式系統 (Hierarchical Styling)**：實作「單張卡片 > 使用者預設 > 系統底層」三層樣式覆蓋機制，確保字體及顏色能被精準繼承。
*   **物理視覺張貼感**：新建卡片上方隨機生成半透明、可能微翹或帶髒污的膠帶效果。

### 🔄 UX 防呆 (Failsafe UX)
*   將垃圾桶圖示隱藏邏輯改為「長時懸停才顯示」，杜絕誤刪。
*   加入刪除/蓋章後的 `Toast` 倒數後悔挽救系統 (Undo 機制)，提供 4 秒無痛復原的防護網。
