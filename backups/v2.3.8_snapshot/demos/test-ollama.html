🤖 Ollama 連線防堵與支援度測試

⚠️ 解決瀏覽器 CORS 跨域限制設定步驟：
前端瀏覽器預設會阻擋對 http://localhost:11434 的請求，因此你必須設定 Windows 環境變數：

1. 退出原本正在運行的 Ollama（去右下角系統匣對 Ollama 點右鍵選 Quit）
2. 按 Win鍵 搜尋「環境變數」(Edit the system environment variables)
3. 點擊 [環境變數] -> 在下半部「系統變數」點 [新增]
4. 變數名稱填入：OLLAMA_ORIGINS ， 變數值填入：*
5. 按確立存檔後，重新啟動 Ollama 軟體。
完成後，點擊下方按鈕測試即可確認技術問題是否已解決！
