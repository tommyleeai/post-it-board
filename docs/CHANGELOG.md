# Changelog (版本更新日誌)

所有關於「Post It 數位白板」的重要變更都會依照時間順序記錄在這個檔案中。

本專案將盡可能遵循 [Semantic Versioning](https://semver.org/spec/v2.0.0.html) 的精神進行版號管理。


































## [2.1.32] - 2026-04-18

### ✨ 新增 (Added)
*   **頂部膠帶拖曳**：解除膠帶的穿透屬性，現在所有便利貼（包含滿版的 YouTube 純影片卡片）都可以直接按住上方「膠帶」來進行拖曳，保持無邊框美感的同時大幅提升操作體驗。

## [2.1.31] - 2026-04-18

### 🔧 優化與修正 (Improved & Fixed)
*   **純影片卡片比例優化**：修復純 YouTube 影片卡片底部會出現大片底紙留白的問題，自動適應 16:9 完美比例。

## [2.1.30] - 2026-04-18

### 🔧 優化與修正 (Improved & Fixed)
*   **YouTube 影片體驗優化**：
    *   修正 YouTube 全螢幕縮放時導致的便利貼排版與圖釘連線跑版問題 (防護 Safari 縮放 Bug)。
    *   取消預設靜音設定，讓無自動播放的影片維持預設音量。
    *   放寬網址辨識條件，支援只複製 `www.youtube.com/...` 的網址也能準確變身為純影片卡片。

*   **交錯流體飛出動畫 (Staggered Silky Animation)**：印章選項套用特製的蘋果級快出慢停曲線 `cubic-bezier(0.2, 0.9, 0.2, 1)`，並配合 `0.05s` 的時間差依序交錯飛出，總耗時極限壓在 0.45 秒以內，帶來不拖泥帶水、極致高級的流體 UI 反饋手感。

*   **印章預覽白底修復**：針對去背景的印章圖案（如「完成」、「准奏」），修復了它們在彈出選單預覽與游標跟隨階段，因為透明背景而導致圖示與軟木塞底圖重疊無法辨識的 Bug。現已皆附上不超出邊框範圍的實心純白底色。
*   **真實印泥覆蓋效果調整**：大幅優化了將印章蓋在便利貼時的物理視覺感，將印章尺寸暴力放大一倍，並讓按壓動畫最終落於 `0.45` 的半透明度，完美重現真實墨水印泥微透紙背，卻又霸氣十足的質感。
