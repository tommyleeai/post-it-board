# AI 語意解析功能 Todo List

> 最後更新：2026-05-14

---

## 🔥 高價值 — 直接強化現有體驗

- [ ] **#1 🔗 智慧連結預覽**
  偵測到 URL 時自動抓取網頁標題、縮圖、描述，產生 Link Card（類似 Slack/Discord 預覽卡片）
  範例：`https://www.youtube.com/watch?v=xxx` → 自動產生預覽卡

- [ ] **#2 🏷️ 自動分類標籤**
  AI 分析內容後自動打上標籤（工作、生活、財務、學習…），可用來篩選/搜尋
  範例：`明天要繳信用卡費` → 自動標籤 `💰財務`

- [ ] **#3 📊 數字智慧卡片**
  偵測到匯率、溫度、單位換算等數字意圖時，自動顯示換算結果
  範例：`100 USD 換台幣` → 即時顯示匯率卡片

- [/] **#4 🎯 待辦事項提取**
  從長段文字中自動提取出多個 action items，變成可勾選的 checklist
  範例：`開會結論：修改報告、聯繫客戶、更新文件` → 可勾選清單

- [ ] **#5 🌍 天氣卡片**
  偵測到地點+出行意圖，自動拉取該地天氣預報
  範例：`週末去台中玩` → 顯示台中週末天氣

---

## ⚡ 中等價值 — 提升智慧感

- [ ] **#6 🔄 卡片關聯推薦**
  AI 發現兩張卡片內容相關時，顯示「相關卡片」提示
  範例：卡片 A `TSLA 股票` + 卡片 B `特斯拉財報日` → 關聯提示

- [ ] **#7 📅 行程衝突視覺化**
  將衝突偵測升級成視覺化時間軸，在白板上用線條連接同一天的行程
  範例：多張行程卡片 → 自動排出時間線

- [ ] **#8 🧮 簡易計算機**
  偵測到數學算式自動計算結果並顯示
  範例：`450 * 12 + 380` → 顯示 `= 5,780`

- [ ] **#9 🌐 多語翻譯**
  偵測到外語內容，自動在卡片底部顯示翻譯
  範例：`Meeting at 3pm tomorrow` → 附註「明天下午3點開會」

- [ ] **#10 📍 地圖卡片**
  偵測到地址/地點名稱，自動嵌入 Google Maps 小地圖
  範例：`台北101` → 顯示迷你地圖

---

## 🧪 創新型 — 差異化殺手功能

- [ ] **#11 🤖 AI 追問對話**
  把卡片變成迷你聊天視窗，AI 主動追問細節（多輪對話）
  範例：`幫我規劃東京五天行程` → AI 追問預算、人數、偏好

- [ ] **#12 📸 圖片語意解析**
  拍照/貼圖到卡片時，用 Gemini Vision 辨識內容（名片→聯絡人、收據→記帳）
  範例：貼一張名片圖片 → 自動提取姓名電話

- [ ] **#13 🔁 週期性摘要**
  每天/每週自動產生一張「AI 摘要卡片」，總結白板上所有待辦進度
  範例：每週一早上自動生成本週待辦摘要

- [ ] **#14 ⚡ 快捷指令系統**
  用 `/` 開頭觸發特殊指令
  範例：`/stock AAPL` 直接建立股票卡、`/timer 25min` 番茄鐘

- [ ] **#15 🧠 情緒偵測**
  偵測卡片內容的情緒（壓力大、開心、焦慮），白板角落顯示「今日心情儀表板」
  範例：`又要加班...` → 😰 壓力指標上升
